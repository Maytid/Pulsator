# Pulsator

### A mod that allows users to pulsate/lerp their P1 and P2 colors between 2 different (or same) colors!!

#### This mod comes with:
- Pulse Speed
- P1 Primary Colors
- P1 Secondary Colors
- P1 Wave Trail Colors
- P2 Primary Colors
- P2 Secondary Colors
- P2 Wave Trail Colors
